    Lease Abstraction API

    Extract structured data from commercial lease PDFs/DOCXs using the AI pipeline (Extractor → Validator → Critic).

    POST /api/extract — AI clause extraction via OpenRouter
    Request / Response 
    // Request headers
Content-Type: application/json
Authorization: Bearer <your-api-key>

// Request body
{
  "system": "You are an attorney-grade lease abstraction AI...",
  "messages": [{ "role": "user", "content": "<lease text here>" }],
  "task": "extraction",
  "model": "anthropic/claude-sonnet-4-5",
  "max_tokens": 16000
}

// Response 200
{
  "content": [{ "type": "text", "text": "{ \"parties\": {...}, \"clauses\": {...} }" }],
  "usage": { "input_tokens": 4200, "output_tokens": 3800 },
  "_meta": { "model": "anthropic/claude-sonnet-4-5", "task": "extraction" }
}

// Response 400 (error)
{ "error": "OpenRouter API key not configured in db/api_config.json" }
    POST /api/extract-text — Server-side PDF text extraction (pdfplumber)
    Request / Response 
    // Request body (base64-encoded PDF)
{
  "fileData": "<base64-encoded PDF bytes>"
}

// Response 200
{
  "success": true,
  "text": "LEASE AGREEMENT\nThis agreement dated...",
  "pages": 12,
  "method": "pdfplumber",
  "chars": 48320
}

// Response 503 (pdfplumber not installed)
{ "error": "pdfplumber not installed. Run: pip install pdfplumber --break-system-packages" }
    POST /api/critique — Critic Agent review (Claude Opus 4.7)
    Request / Response 
    // Request body
{
  "extracted": { "parties": {...}, "clauses": {...} },
  "flags": [{ "id": "parties.tenant_name", "severity": "critical" }],
  "leaseText": "LEASE AGREEMENT\nThis agreement..."
}

// Response 200
{
  "operations": [
    { "field": "parties.tenant_name", "action": "set", "value": "Oracle Flooring Direct LLC" }
  ],
  "unresolved": []
}

    Translation API

    Translate documents between languages using OpenAI models. All translation requests go through the same /api/extract proxy with "task":"quick" and an OpenAI model.

    POST /api/extract — Translation via OpenAI (task: "quick")
    Request / Response 
    // Request body
{
  "system": "You are a professional document translator. Translate the following text to Spanish. Preserve all formatting, headers, and structure. Return only the translated text.",
  "messages": [{ "role": "user", "content": "LEASE AGREEMENT\nThis agreement..." }],
  "task": "quick",
  "model": "openai/gpt-4o-mini",
  "max_tokens": 16000
}

// Response 200
{
  "content": [{ "type": "text", "text": "CONTRATO DE ARRENDAMIENTO\nEste acuerdo..." }],
  "usage": { "input_tokens": 1200, "output_tokens": 1350 },
  "_meta": { "model": "openai/gpt-4o-mini", "task": "quick" }
}

// Supported models
// Fast:   openai/gpt-4o-mini       (default translation)
// Better: openai/gpt-4o            (high-accuracy mode)
// Legal:  anthropic/claude-sonnet-4-5  (legal document translation)
